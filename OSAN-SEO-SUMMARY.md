## 1분 요약

**오산역·세교신도시 중심의 新 가라오케/하이퍼블릭 가이드 사이트**.
신규 도메인이므로 3개월 집중 전략이 필요.
기존 사이트(분당, 정자, 기흥)의 성공 모델을 재적용 + 지역 특화.

**목표**: 
- **3개월**: 월 3,000-5,000 세션, "오산 가라오케" Top 10 진입
- **6개월**: 월 5,000-8,000 세션, "오산 하이퍼블릭" Top 5 진입
- **연간**: 월 10,000+ 세션, 상담 400+ 건/월

## 핵심 전략 (3가지)

### 1. 콘텐츠 중심 (Content-First)
- **기본**: 메타 최적화 + 기본 요약 (Week 1)
- **확대**: 블로그 주 2회 포스트 (26개/3개월)
- **심화**: 신규 페이지 개발 (8개 + 블로그)

### 2. 로컬 최적화 (Local-First)
- **Google Business Profile**: 즉시 활성화
- **네이버 플레이스**: 즉시 등록 + 적극 관리
- **인용(Citation)**: Kakao Map, 당근마켓 등 등록

### 3. 신뢰도 강화 (Authority-Building)
- **내부 링크**: 기존 사이트 네트워크 활용
- **리뷰**: 50개 이상 확보 (4.5+ 평점 유지)
- **백링크**: 기존 지역 사이트에서 링크

## 즉시 실행 (Week 1)

### 30분 안에
```
1. Google Business Profile 기본 정보 입력
   └─ 비즈니스명, 전화, 주소, 링크

2. 네이버 플레이스 등록 시작
   └─ 기본 정보 + 사진 10장

3. Google Search Console 사이트맵 제출
   └─ https://osankaraoke.com/sitemap.xml

4. robots.txt 확인
   └─ 색인 차단 없는지 확인
```

### 1-2시간 안에
```
5. 메타 데이터 스프레드시트 작성
   ├─ 6개 페이지 (셔츠룸, 룸살롱, 기모노, 호빠, 가격, 초보자)
   └─ 각 페이지별 Title, Description, Keywords

6. 상단 요약 템플릿 작성
   └─ 각 업종 페이지 H2 아래 추가

7. 내부 링크 구조 검토
   └─ 관련 페이지 링크 확인
```

## 전환율 최적화 (CRO)

### 현재 강점
```
✓ 명확한 CTA (전화, 카카오톡, 텔레그램)
✓ 신뢰도 신호 (15년 경력, 만족도 4.8/5.0)
✓ 다양한 서비스 소개
✓ 하단 고정 CTA 버튼 (모바일)
```

### 개선 기회
```
[ ] A/B 테스트 1: CTA 버튼 색상/텍스트
    └─ 목표: 클릭율 +20%

[ ] A/B 테스트 2: 메인 헤딩 변형
    └─ 옵션 1: "오산 NO.1 유흥 가이드"
    └─ 옵션 2: "투명 가격, 안심 예약"

[ ] 리뷰 추가 (고객 사진 포함)
    └─ 목표: 50개 이상 확보

[ ] CTA 버튼 크기 조정
    └─ 모바일: 1.2배 확대
    └─ 클릭 영역: 최소 44x44px
```

```

ROI: 예상 상담 300-400건 × 평균 수익 = 충분히 회수
```
