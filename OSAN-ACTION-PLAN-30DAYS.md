### Day 2-3: 기본 요약 추가 (Priority: P1)
**담당**: 콘텐츠 담당자 | **예상 시간**: 3시간

      {highlights.map((item, idx) => (
        <div key={idx} className="text-amber-400">

);

## 문제 해결 가이드

### 문제 1: 블로그 포스트 작성 지연
```
증상: 예정된 날짜에 포스트 게시 못함
해결:
1. 템플릿 미리 준비 (대략 구조 파악)
2. 문장 구성 단순화 (너무 길지 않게)
3. 필요시 포스트 1개를 2주로 늘임 (우선순위 조정)
4. 외부 필자 또는 도움 요청 검토
```

### 문제 2: 메타 데이터 적용 안 됨
```
증상: 메타 데이터 입력했는데 Google에 반영 안 됨
해결:
1. 48-72시간 대기 (Google 크롤링 시간)
2. Google Search Console에서 "URL 검사" → "색인 생성 요청"
3. HTML 소스에서 실제로 적용되었는지 확인
4. 캐시 문제 시 hard refresh (Cmd+Shift+R)
```

### 문제 3: 순위 상승 안 함
```
증상: 콘텐츠 작성했는데 순위 변화 없음
해결:
1. 2-4주 대기 (색인 생성 + 평가 시간)
2. Google Search Console에서 실제 색인 여부 확인
3. 콘텐츠 품질 점검 (너무 짧지는 않은지, 키워드 포함 여부)
4. 백링크 확인 (도메인 권위성 부족할 수 있음)
5. 경쟁사 콘텐츠와 비교 (더 나은 콘텐츠 필요?)
```
